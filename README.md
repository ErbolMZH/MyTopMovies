# MyTopMovies
Здесь вы можете найти рейтинговые и новые фильмы. 
Основные использованные  технологий:
  1. Java SE
  2. Adapters
  3. JSON
  4. Получил данные из сайта https://api.themoviedb.org/3/discover/movie
  5.SQLLite6 DAO
